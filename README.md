##  Installation Flow  

1. Download and run the `MusicPlayerSystemSetup.exe` file.  
2. Follow the on-screen setup wizard.  
3. Once installed, your browser will open the **Thank You** page.  
4. If uninstalled, the **Goodbye** page will appear automatically.  

This provides a **real-world professional app experience** similar to commercial software installations.
